# The Coal City Heist — NC Enugu 2026

**Pressure Makes Diamonds.**

Conference registration site for AIESEC NC Enugu 2026. Built with React + Vite.

## Setup

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub → connect repo to [Vercel](https://vercel.com) → deploy. Zero config needed.

## Stack

- **React** — UI
- **Vite** — build tool
- **Vercel** — hosting
- **Supabase** *(planned)* — delegate data, quiz results, case numbers
- **Paystack** *(planned)* — payments
